## 📘 프로젝트 개요

**부서 금전 관리 시스템**는 (주)선진 디지털SCM팀의 하계 현장실습으로 진행된 프로젝트입니다.<br>
사용자가 지출 항목을 신청하고, 지정된 결재 라인에 따라 승인 절차가 진행되는 기능을 제공합니다.<br>
결재 라인을 통해 체계적으로 결재되어 투명하고 일관된 관리를 지원합니다.
<br>

## 👥 팀 구성

| 역할             | 이름     |
|------------------|----------|
| **백엔드 개발**  | **김세윤**   |
| 프론트엔드 개발 | 이규태   ||
<br>

## 🛠 기술 스택

- **언어**: Java
- **프레임워크**: Spring Boot
- **데이터베이스**: PostgreSQL
- **API 문서**: Swagger
<br>

## 📌 주요 기능

- JWT 기반 회원가입 및 로그인 (Redis를 활용한 액세스/리프레시 토큰 관리)
- 지출 항목 신청서 작성, 수정 및 삭제
- 사용자별 결재 라인 설정
- 결재 단계별 승인·반려 처리 및 상태 이력 추적
